Claude Code

终端里的 Claude，指向同一个网关。

一行配置，把 Claude Code 的流量切到 Mer3y。安装方式不变，日常开发节奏不变。

PowerShell Node.js Homebrew
install 
irm https://metapi.mer3y.xyz/install.ps1 | iex
npm install -g @anthropic-ai/claude-code
brew install claude-code
~/.claude/settings.json 
{
  "env": {
    "ANTHROPIC_AUTH_TOKEN": "your-api-key",
    "ANTHROPIC_BASE_URL": "https://metapi.mer3y.xyz/v1"
  }
}

OpenAI / Codex CLI

Codex CLI，同一个 base_url。

Responses API 直连，配置写在本地 TOML 里。模型和鉴权各管各的，改一行切一个。

Responses API TOML Bearer Token
~/.codex/config.toml 
model_provider = "aether"
model = "latest-model-name"
model_reasoning_effort = "high"

[model_providers.aether]
name = "OpenAI"
base_url = "https://metapi.mer3y.xyz/v1"
wire_api = "responses"
requires_openai_auth = true
auth.json 
{
  "OPENAI_API_KEY": "your-api-key"
}

Gemini CLI

Gemini CLI，三个环境变量。

环境变量式接入，适合脚本和临时工作区。BASE_URL + Key + Model，终端即走。

3 个变量 1 个入口 CLI 友好
.env 
GOOGLE_GEMINI_BASE_URL=https://metapi.mer3y.xyz/v1
GEMINI_API_KEY=your-api-key
GEMINI_MODEL=latest-model-name
